## Mario-maker-2-pixel  
is a remake of mario maker but in pixel form!
 

> Open this page at [https://lamannamamma.github.io/mario-maker-2-pixel/](https://lamannamamma.github.io/mario-maker-2-pixel/)

## Use as Extension

This repository can be added as an **extension** in MakeCode.

* open [https://arcade.makecode.com/](https://arcade.makecode.com/)
* click on **New Project**
* click on **Extensions** under the gearwheel menu
* search for **https://github.com/lamannamamma/mario-maker-2-pixel** and import

## Edit this project

To edit this repository in MakeCode.

* open [https://arcade.makecode.com/](https://arcade.makecode.com/)
* click on **Import** then click on **Import URL**
* paste **https://github.com/lamannamamma/mario-maker-2-pixel** and click import
